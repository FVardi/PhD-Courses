USAGE QUICKSTART
================================================================================

BASIC USAGE:
  from pathlib import Path
  from scripts.task01_eda import DataLoader

  loader = DataLoader(data_root=Path('data/london_smart_meters'))

LOAD A SINGLE BLOCK:
  df = loader.load_block('daily_dataset', 0)

LOAD MULTIPLE BLOCKS:
  df, metadata = loader.load_and_clean_dataset(
      dataset_type='daily_dataset',
      blocks=[0, 1, 2],
      sort=True,
      handle_duplicates='keep_first'
  )

LOAD ALL BLOCKS:
  df, metadata = loader.load_and_clean_dataset(
      dataset_type='halfhourly_dataset',
      blocks=None,  # None = all 112 blocks
      sort=True,
      handle_duplicates='resolve'
  )

GET STATISTICS:
  summary = loader.get_summary_statistics(df, 'daily_dataset')
